Сгенери картинку: $ARGUMENTS

Используй skill `content-gen`.

## Auto-route

1. **Если $ARGUMENTS ≤ 5 значимых слов** (без артиклей/предлогов) — переходи в **brief-режим** (`/content-brief` flow): прочитай `~/.claude/skills/content-gen/lib/prompt-engineering.md`, контекст проекта (если есть), задай 3-5 вопросов через `AskUserQuestion`, собери enriched prompt, покажи на подтверждение.

2. **Если $ARGUMENTS детальный** (≥ 6 значимых слов с указанием стиля/настроения/контекста) — запускай **сразу**:
   ```bash
   ~/.claude/skills/content-gen/run.sh "<prompt>"
   ```
   Default tier = `mid` (3 источника, ~$0.06).

## Tier hints (как выбирать)

- Если пользователь сам пишет «попробуй / прикинь / посмотрим что выйдет» → `--tier explore` (~$0.04)
- Если в контексте «hero / финал / для production / на сайт» → `--tier premium` (~$0.14)
- Иначе → `--tier mid` (default)

## Russian → English

Если $ARGUMENTS на русском — **сам переведи на английский** перед вызовом скрипта (image-AI лучше работают на EN). Сохраняй оригинал — покажи пользователю «оригинал → translated» если он переводился, чтобы убедился что смысл сохранён.
